Atividade Prática - Angular
Este projeto demonstra a aplicação dos conceitos básicos do Angular através de um componente de contador.

Conceitos Aplicados:
Interpolação {{ }}: Exibição do título e do valor do contador.

Property Binding [ ]: Vinculação da URL da imagem ao atributo src.

Event Binding ( ): Evento de clique nos botões para alterar o valor.

Diretiva *ngIf: Condição para mostrar um aviso se o valor for negativo.

Diretiva *ngFor: Lista o histórico de alterações na tela.
